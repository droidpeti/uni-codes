Binary search tree

If for every element in the tree the element left from the parent is smaller, the element right from the parent is larger.
When we allow for elements to be equal, that is a binary sort tree, not a search tree

tree *search_tree(tree *l, item_type x){
    if(l == NULL){
        return NULL;
    }

    if(l->item == x){
        return l;
    }
    if(x < l->item){
        return search_tree(l->left, x);
    }
    else{
        return search_tree(l->right, x);
    }
}
